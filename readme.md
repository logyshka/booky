### 1. Технические требования

Python 3.11.0

### 2. Установка

###### Устанавливаем зависимости

```bash
pip install -r r.txt
```

###### Создаём миграции <small>(фу чурки)</small>

```bash
python manage.py makemigrations
```

###### Применяем созданные миграции <small>(снова чурки)</small>

```bash
python manage.py migrate
```
